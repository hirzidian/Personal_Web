My Skills

PHP

PHP, singkatan dari "Hypertext Preprocessor," adalah bahasa pemrograman sumber terbuka untuk pengembangan aplikasi web. Sebagai bahasa skrip server, PHP dijalankan di server, menghasilkan HTML yang dikirimkan ke browser pengguna.

Laravel

Laravel adalah kerangka kerja (framework) PHP yang dirancang untuk mempermudah pengembangan aplikasi web dengan menyediakan struktur kerja yang konsisten, sintaksis ekspresif, dan berbagai fitur bawaan yang dapat digunakan oleh pengembang.

Bootstrap

Bootstrap adalah kerangka kerja (framework) front-end sumber terbuka yang dikembangkan oleh Twitter. Tujuannya adalah untuk mempermudah pengembangan situs web responsif dan mobile-friendly.

JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi yang bersifat dinamis dan sering digunakan untuk mengembangkan aplikasi web interaktif. JS bersama dengan HTML dan CSS merupakan salah satu teknologi inti untuk pengembangan web.

Vue Js

Vue.js adalah kerangka kerja JavaScript progresif untuk membangun antarmuka pengguna aplikasi web. Dengan desain ringan dan integrasi mudah, Vue.js memungkinkan pembuatan komponen-komponen yang dapat digunakan kembali, memastikan responsivitas aplikasi.

NodeJs

Node.js adalah sebuah runtime lingkungan (runtime environment) yang memungkinkan eksekusi kode JavaScript di sisi server. Dirancang untuk menangani aplikasi yang memerlukan kinerja tinggi dan dapat menangani banyak koneksi secara bersamaan.
